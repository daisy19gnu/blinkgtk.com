# blinkgtk.com

[blinkgtk.com](https://blinkgtk.com) の公開サイト（GitHub Pages でホスティング）。

BlinkGTK の**商用窓口**です — 製品での採用、PoC、受託開発のご相談を扱います。
技術・コミュニティ情報は [blinkgtk.org](https://blinkgtk.org) を参照。

## 構成
- `index.html` — トップページ（最小プレースホルダ）
- `CNAME` — カスタムドメイン設定（`blinkgtk.com`）

## デプロイ
`main` ブランチへの push で GitHub Pages が自動公開します。
